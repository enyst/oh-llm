# Provider references

Quick links + common “gotchas” for major LLM providers when used via **LiteLLM** (which presents an OpenAI-style API).

These notes are intentionally short and point to upstream docs. Treat provider docs + LiteLLM docs as the source of truth.

## Index

- `openai.md`: OpenAI API
- `anthropic.md`: Anthropic Claude API
- `gemini.md`: Google AI Studio / Gemini API (including OpenAI compatibility endpoint)
- `openrouter.md`: OpenRouter (OpenAI-compatible proxy/marketplace)
- `groq.md`: Groq (OpenAI-compatible endpoint)
